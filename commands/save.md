# Save Session Progress

Read the "Branch Memory" section of CLAUDE.md to understand how log and state files work in this project.

Determine the current git branch. Then update the branch memory for this branch:

- Update the **state file** to reflect the current state of the work
- Update the **log file** with what was done in this session

Use your knowledge of the current session to populate both files. If the files don't exist yet, create them.
