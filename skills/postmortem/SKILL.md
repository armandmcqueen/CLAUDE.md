Guide the user through each section conversationally — ask questions one or two at a time, not all at once. Build the postmortem incrementally through dialogue. Once all sections are complete, compile the final document.

Save the postmortem to `claude/postmortems/` with the naming convention `YYYY-MM-DD-<short-description>.md`.
